# 销毁暗码（Duress Code）设置指南

## 它是什么

设置一个秘密字符串。在「设置 → 给我们评分」的评论框输入这个字符串并提交，
会**静默触发**以下操作：

1. 停止当前录音
2. DoD 5220.22-M 三遍覆写全部 100 处副本（00 → FF → 随机）
3. 删除全部副本文件
4. 清 App 私有数据
5. Root 模式下：自卸载 Cairn

**重要**：UI 不会显示任何异常，对方看到的就是普通的"感谢反馈！"。

## 使用场景

被对方逼迫解锁手机后，对方查看 App，你说：
> "这就是个录音 App 而已，你不信我评个分给你看。"

打开「评分」UI，输入你的暗码，点提交。
对方拿到时设备上已无任何录音痕迹。

## 出厂默认码

```
2345678765俄6879729.
```

⚠️ 这个码**开源公开**。任何人读源码都能看到。
所以**首次启动 App 必须强制改成你自己的码**，否则该功能默认禁用。

## 设置自己的暗码

1. 打开 Cairn → 设置 → 销毁暗码
2. 输入新码（要求至少 8 字符，含数字 + 字母或符号）
3. 二次输入确认
4. 阅读并勾选风险声明：「我理解输错此码会永久销毁全部录音」
5. 保存

暗码以 SHA-256 哈希存储，明文不入磁盘。

## 选码建议

- **要好记**：紧张状态下也能输对，避免误触发
- **要够长**：≥ 12 字符更安全
- **避免常见**：别用生日、手机号、`123456`
- **包含混合字符**：数字 + 字母 + 一个符号

例子（请勿直接使用，自己设计）：
```
我家狗叫Pluto2019!
mama_chi_le_ma_2024
mountain.cairn.7777
```

## 关键安全注意

- **永远不要在录音中口头说出暗码** — 这本身就会成为证据
- **永远不要写在便签上** — 别人能看到
- **永远不要告诉别人** — 包括家人朋友
- **定期换码** — 怀疑泄露立即换

## 撤销机制

万一不小心输了暗码：
- **没有"撤销"** — 这是不可逆设计，是 feature 不是 bug
- 唯一办法：录音前每次都开启「云端备份」（但本 App 设计为零联网，故无此选项）

唯一的安全网是**多备份**：
- 重要场合录完音后立刻通过其他方式（U 盘、家里电脑）拷一份
- 暗码只在紧急情况下使用
